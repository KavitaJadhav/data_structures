Here are **clean text notes for Graph Terms** (easy to revise for interviews).

---

GRAPH BASICS

Graph = A collection of **vertices (nodes)** connected by **edges**.

G = (V, E)

V = vertices
E = edges

Example:

A — B
|   |
C — D

Vertices = A, B, C, D
Edges = (A,B), (A,C), (B,D), (C,D)

---

VERTEX (NODE)

A point in a graph.

Example:
A, B, C

---

EDGE

Connection between two vertices.

Example:
(A,B)

---

DIRECTED GRAPH (DIGRAPH)

Edges have direction.

A → B → C

Edge (A,B) means A can go to B.

---

UNDIRECTED GRAPH

Edges have no direction.

A — B

Both nodes are connected.

---

WEIGHTED GRAPH

Edges have weights (costs).

A --5-- B
B --3-- C

Used in shortest path problems.

---

DEGREE OF A VERTEX

Number of edges connected to a vertex.

Example:

A — B — C
|
D

deg(B) = 3

---

IN-DEGREE (Directed Graph)

Number of incoming edges.

Example:

A → B ← C

in-degree(B) = 2

---

OUT-DEGREE (Directed Graph)

Number of outgoing edges.

Example:

A → B → C

out-degree(B) = 1

---

PATH

Sequence of vertices connected by edges.

Example:

A → B → C → D

Path length = number of edges.

---

CYCLE

A path where start and end node are the same.

Example:

A → B → C → A

---

CONNECTED GRAPH

Every vertex is reachable from every other vertex.

Example:

A — B — C

---

DISCONNECTED GRAPH

Some nodes cannot be reached.

Example:

A — B     C — D

---

COMPONENT

A connected subgraph inside a graph.

Example:

A — B     C — D

Components = 2

---

TREE

Special graph with:

* no cycles
* connected
* n nodes → n-1 edges

Example:

```
A
```

/
B   C

---

DAG (DIRECTED ACYCLIC GRAPH)

Directed graph with **no cycles**.

Example:

A → B → D
A → C → D

Used in scheduling and dependency graphs.

---

GRAPH REPRESENTATION

Adjacency List

A → B, C
B → A, D
C → A
D → B

Space Complexity

O(V + E)

---

Adjacency Matrix

```
A B C
```

A [ 0 1 1 ]
B [ 1 0 0 ]
C [ 1 0 0 ]

Space Complexity

O(V²)

---

GRAPH TRAVERSAL

BFS (Breadth First Search)

Uses **Queue**

Visits nodes **level by level**

Applications:

* shortest path (unweighted)
* social networks

---

DFS (Depth First Search)

Uses **Stack or Recursion**

Explores **deep paths first**

Applications:

* cycle detection
* topological sort

---

GRAPH COMPLEXITY

BFS

Time = O(V + E)

DFS

Time = O(V + E)
